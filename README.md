# rbfdiff
MATLAB code for differentiation of RBF approximations both using the direct method and using the stable RBF-QR method.

If you use the code, please cite the following paper:

> E. Larsson, A. Heryudono, A. Michael, and C. Piret, *A Matlab code for stable differentiation of radial basis function approximations*, J. Approx. Softw., 3(2026), pp. 56-76. [https://doi.org/10.13135/3103-1935/12293](https://doi.org/10.13135/3103-1935/12293)

and relevant references therein if appropriate.
